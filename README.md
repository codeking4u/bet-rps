## Technologies used:

1. React with TypeScript for UI.
2. Context API for state management.
3. CSS Modules(With SASS) for styling.
4. Jest for testing.

## Running the Application:

1. Install all dependencies

### `npm install`

2. Start the Application

### `npm start`

3. For testing

### `npm run test`

4. For building

### `npm run build`
